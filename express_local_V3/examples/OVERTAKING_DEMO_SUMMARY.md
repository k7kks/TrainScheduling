## 🔧 技术实现

### 越行判定条件

```python
# 当满足以下条件时，判定需要越行
if express_arrival < local_departure + min_tracking_interval:
    # 快车会追上慢车，需要越行
    overtaking_required = True
```

### 越行处理逻辑

```python
# 1. 计算慢车新的发车时间
new_local_departure = express_pass_time + min_pass_departure_interval

# 2. 计算慢车停站时间
new_dwell_time = new_local_departure - local_arrival_time

# 3. 确保停站时间≥240秒
if new_dwell_time < min_overtaking_dwell:
    new_dwell_time = min_overtaking_dwell
    new_local_departure = local_arrival_time + new_dwell_time

# 4. 顺延后续站点
time_shift = new_local_departure - original_local_departure
for stop in subsequent_stops:
    stop.arrival_time += time_shift
    stop.departure_time += time_shift
```
