4. 查看站台154、172等
5. 应该能看到明显的水平停留（4-5分钟）

## ⚠️ 常见问题

### Q1: 运行图中看不到明显的越行？

**A**: 请**缩小时间刻度**！
- 如果刻度太大（如每格10分钟），4分钟的停留看起来很短
- 建议：每格1-2分钟

### Q2: 站台码还是不对？

