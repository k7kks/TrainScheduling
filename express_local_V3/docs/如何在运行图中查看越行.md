## 🔍 如何查看越行

### 方法1：查看Excel数据（最直接）

1. 打开`result.xlsx`
2. 查看"计划车次数"sheet页
3. 找到慢车车次（例如车次33、41、43、44、49等）
4. 查看这些车次的详细时刻表

**验证方式**：
- 对比同一车次在不同站的停站时间
- 正常站：停站约30秒
- **越行站：停站270-300秒** ← 明显增加！

### 方法2：在运行图软件中缩小时间刻度

如果您的运行图看起来所有线条都是斜线，没有水平停留：

**原因**：时间刻度太大，240秒（4分钟）的停留看起来不明显

**解决方案**：
1. 找到运行图软件的时间刻度设置
2. 将刻度从"每格10分钟"缩小到"每格2分钟"或"每格1分钟"
3. 放大查看越行站附近的区域

**预期效果**：
- 缩小刻度后，慢车线在越行站会有明显的水平停留（4-5分钟）
- 快车线在此时通过（斜线穿过慢车的水平停留）

### 方法3：手动计算验证

选择一列被越行的慢车（例如车次33）：

1. 找到它的完整时刻表
2. 查看站台142的到发时间
3. 计算停站时间 = 发车时间 - 到达时间

**示例**：
```
车次33在站台142:
  到达时间: 07:05:10 (25510秒)
  发车时间: 07:09:40 (25780秒)
  停站时间: 270秒（4分30秒）← 这就是越行！
```

对比车次33在其他站的停站时间：
```
站台114: 停站30秒（正常）
站台154: 停站30秒（正常）
站台142: 停站270秒（越行！）← 明显增加
站台162: 停站30秒（正常）
```

### 方法4：对比快慢车在同一站的时刻

选择一对越行的快慢车（例如快车X和慢车33在站台142）：

| 列车 | 到达时间 | 发车时间 | 说明 |
|------|----------|----------|------|
| 慢车33 | 07:05:10 | 07:09:40 | 停站270秒（等待快车） |
| 快车X | 07:07:30 | 07:07:30 | 通过（不停站） |

**观察**：
- 慢车到达：07:05:10
- 快车通过：07:07:30（在慢车等待期间）
- 慢车发车：07:09:40（快车通过后）
- **这就是越行！**

## 💡 如果还是看不到越行

### 调整1：创建更明显的越行场景

修改`run.py`参数，创建更极端的越行：

```python
"--express_ratio", "0.2",      # 只有20%快车（更少快车，更明显）
"--target_headway", "300",     # 发车间隔5分钟（更大间隔）
```

然后在`timetable_builder.py`中增加越行停站时间：

```python
self.min_overtaking_dwell = 480  # 增加到8分钟（非常明显）
```

### 调整2：使用简单测试场景

运行简单测试：

```bash
python express_local_V3/examples/overtaking_demo.py
```

这个demo只有3列快车和4列慢车，越行现象更明显。
